How to host your own clan site

  1. Download the latest release from github
  2. Fill in the blanks for Clan name
  3. Fill in some clan statistics
  4. Describe your clan for the about page
  5. insert your own action_page.php to handle input from the contact page
  6. host the pages, images, and css
  7. tell your clan mates where they can find the site!
